**No Crosshair is a lightweight client-side mod that completely removes the default crosshair from your Minecraft HUD.**

**Perfect for cinematic videos, screenshots, or players who prefer a minimalist interface, this mod helps you enjoy a clean and immersive view of the game world.**

![preview](https://i.postimg.cc/PJhvpXgF/image-2025-09-24-145153204.png)

- ✦ Works seamlessly with most shaders and resource packs.
- ✦ No configuration needed — just install and play.
- ✦ Compatible with the latest versions of Minecraft.
